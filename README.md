# ProfNavigator

Система профориентации с ML-классификацией результатов. Определяет подходящие профессиональные сферы на основе ответов пользователя.

## Структура проекта

```
project/
├── main.py              # FastAPI сервер
├── model.py             # ML модель (загрузка и предсказание)
├── trainer.py           # Скрипт для обучения модели
├── data_gen.py          # Генерация синтетического датасета
├── questions.json       # Вопросы опроса (10 категорий)
├── dataset.json         # Датасет для обучения
├── responses.json       # Накопленные ответы пользователей
├── model_artifact.pkl   # Обученная модель
├── requirements.txt     # Зависимости
└── static/
    └── index.html       # Веб-интерфейс
```

| **technical** | Инженерия, технические системы ⚙️ |
| **artistic** | Художественное творчество, перформанс 🎭 |
| **entrepreneurial** | Бизнес, предпринимательство, стартапы 💼 |
| **scientific** | Фундаментальная наука, академическая деятельность 🧪 |

## Быстрый старт

### 1. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 2. Запуск сервера

```bash
uvicorn main:app --reload
```

Откройте http://127.0.0.1:8000 в браузере.

## Обучение модели

### Вариант 1: Обучение на готовом датасете

```bash
python trainer.py --model-type random_forest
```

### Вариант 2: Генерация нового датасета

```bash
python data_gen.py
python trainer.py --model-type random_forest
```

### Вариант 3: Переобучение на накопленных данных

После прохождения опросов пользователями данные сохраняются в `responses.json`. Для переобучения:

1. Скопируйте `responses.json` в `dataset.json` (оставив только секцию `samples`)
2. Запустите `python trainer.py --model-type random_forest`

### Доступные типы моделей

| Тип | Описание |
|-----|----------|
| `decision_tree` | Дерево решений (быстрая, интерпретируемая) |
| `random_forest` | Случайный лес (по умолчанию, лучшая точность) |
| `neural_network` | Нейронная сеть MLP (сложные паттерны) |

Пример:
```bash
python trainer.py --model-type neural_network
```

## Формат данных

### dataset.json

```json
{
  "samples": [
    {
      "features": {
        "analytical": 5,
        "social": 1,
        "creative": 1,
        "managerial": 1,
        "practical": 1,
        "research": 1,
        "technical": 1,
        "artistic": 0,
        "entrepreneurial": 1,
        "scientific": 1
      },
      "label": "analytical"
    }
  ]
}
```

Где:
- `features` — баллы по 10 категориям (0–5)
- `label` — целевая категория для обучения

## API Endpoints

| Метод | Endpoint | Описание |
|-------|----------|----------|
| GET | `/` | Главная страница (веб-интерфейс) |
| GET | `/questions` | Получить 5 случайных вопросов |
| POST | `/submit` | Отправить ответы, получить результат |
| GET | `/api/health` | Проверка статуса модели |

### Пример запроса

```bash
# Получить вопросы
curl http://127.0.0.1:8000/questions

# Отправить ответы
curl -X POST http://127.0.0.1:8000/submit \
  -H "Content-Type: application/json" \
  -d '{"answers": [{"question_id": 1, "option_id": "a"}, ...]}'
```

## Как это работает

1. Пользователь проходит опрос (5 случайных вопросов)
2. Ответы подсчитываются по 10 профессиональным категориям
3. ML-модель предсказывает наиболее подходящую сферу
4. Результат с вероятностями и ранжированием показывается пользователю
5. Ответы сохраняются в `responses.json` для будущего переобучения

## Технологии

- **Backend:** FastAPI
- **ML:** scikit-learn (Random Forest, Decision Tree, MLP)
- **Frontend:** HTML/CSS/JS (static/index.html)
